You can interact with Salesforce from an external application. You can make it either using the Salesforce's REST and SOAP APIs to create, update and delete records, or [[Executing Apex Code from external applications]]. Unlike Salesforce APIs, Apex web service methods run with system privileges and don´t respect the user's object and field permissions. You can force sharing rules with the `with sharing` keyword when you are [[Exposing a class as a SOAP web service]] or [[Exposing a class as a REST web service]].
