Puedes hacerlo con `git status`.
```
git status

On branch master
Nothing to commit, working directory clean
```
Indica que tienes un directorio de trabajo limpio, es decir, no hay archivos rastreados o modificados. Además, Git no encuentra archivos sin rastrear, de lo contrario los listaría aquí. También te informa en qué rama estás y que no hay cambios respecto a la misma rama remota en el servidor.
