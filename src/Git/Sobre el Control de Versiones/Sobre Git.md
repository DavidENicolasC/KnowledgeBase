En el 2002, el proyecto del Kernel de Linux empezó a usar un [[Sistema de Control de Versiones Distribuido]] propietario llamado BitKeeper.

En el 2005, la relación entre la comunidad a cargo del Kernel de Linux y la compañía que desarrollaba BitKeeper terminó. Esto impulsó a la comunidad que desarrollaba Linux (en particular a Linus Torvalds) a desarrollar su propia herramienta basada en algunas cosas que aprendieron mientras usaban BitKeeper.

Git nació oficialmente en el 2005.
