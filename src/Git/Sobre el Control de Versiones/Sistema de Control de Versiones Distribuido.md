Siglas DVCS.  Los clientes no sólo descargan la última copia de los archivos desde un servidor, sino el repositorio completo.

De esta forma, si el servidor se daña o se pierden los archivos, en el sistema de cada colaborador existirá una copia exacta que puede reemplazarlo perfectamente. 

Inclusive, muchos de estos VCS se encargan de manejar numeros repositorios remotos, de forma que puedes colaborar en más de un servidor.
