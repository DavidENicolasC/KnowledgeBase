Siglas CVCS. Hay un único servidor que contiene todos los archivos y los clientes descargan los cambios desde ese servidor central. 

Esto ofrecía ventajas, como saber hasta qué cierto punto estaban trabajando todos en el proyecto.

Sus desventaja más obvia es el posible fallo que puede tener un único servidor conteniendo todos los archivos. Si por alguna razón se dañaba el servidor o se borraba el proyecto, se perdería por completo.
