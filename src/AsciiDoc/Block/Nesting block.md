[[Block]]s can also be nested inside [[Section block]]s, [[List item]]s and [[Table cell]]s.

For nesting, the parent block must have the [[Compound]] [[Content model]].

When you are using the same [[Structural containers]] for nested block, it's necessary to vary the length of the delimiter lines.

```
====
Dummy text

.Title
[%collapsible]
======
Escape into real world.
======
====
```

Can be either shorter or longer than the parent. That's a personal style choice.
