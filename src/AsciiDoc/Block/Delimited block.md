Bounded by [[Linewise delimiters]]. Is used either to enclose other blocks or set the [[Content model]] of the content. Block doesn´t end until the ending delimiter is found. Cannot be interleaved.

Are defined using [[Structural containers]]. Provides a default context and expected content model, but actual context and content model is determined after considerating the [[Block metadata lines]].

This block can be used for [[Nesting block]]s.
