Use a [[Block ID]].
```
[#tigers-subspecies]
=== Subspecies of Tiger
```

You can also add a custom optional reference text either using the [[Shorthand notation]] for [[Block ID]] or anchors ([[Using anchors for a description list item]]).
```
[#tigers-subspecies,reftext=Subspecies]
=== Subspecies of Tiger

[[tigers-subspecies,Subspecies]]
=== Subspecies of Tiger
```

Value of the [[reftext Attribute]] must be quoted if it contains spaces or commas.

For the anchor syntax, ID must conform to the [[XML Name rules]].
