Content between the [[Part Title]] and the first [[Section]] in the [[Book Part]].

Can be marked explicitly using the  `partintro` style on either a [[Paragraph Text]] or [[Open Block]].

Normally is inferred.

```
= Book Title
:doctype: book

= Part I

This is the implicit partintro.

OR 

[partintro]
This is the implicit partintro.

== Chapter A
```
