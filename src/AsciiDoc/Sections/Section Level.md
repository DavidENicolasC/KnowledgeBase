Must be nested logically.

1. An [[AsciiDoc Document]] can only have multiple level 0 sections if the [[Document Type]] is set to [[book Document Type]].
	1. The first level 0 section is the [[Document Title]]; subsequent level 0 sections represent [[Part Title]]s.

2. Order cannot be skipped when nesting sections (e.g., you can’t nest a level 5 section directly inside a level 3 section; an intermediary level 4 section is required)
