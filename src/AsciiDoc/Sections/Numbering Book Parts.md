[[Book Part]] numbers are controlled by the `partnums` attribute, not `sectnums`.
```
= The Secret Manual
:doctype: book
:sectnums:
:partnums:

= Defensive Operations

== An Introduction to DefenseOps

= Managing Werewolves
```

When rendered, numbers are displayed as Roman numerals.

The [[Signifier Part]] is automatically added to the beginning of the [[Part Title]]s when this attribute is set.
