Used to express gratitude.

Document type must be [[book Document Type]].

must be a level 1 section `==`, unless the book has parts.

```
[dedication]
== Dedication

For S.S.T.--

thank you for the plague of archetypes.
```

If the book has parts, the dedication section must be a level 0 section (=).

```
[dedication]
= Dedication

For S.S.T.--

thank you for the plague of archetypes.
```
