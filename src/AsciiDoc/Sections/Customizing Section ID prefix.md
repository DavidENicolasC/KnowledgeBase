Use the `idprefix` attribute.

Customizing:
```
:idprefix: id_
```

Removing:
```
:idprefix:
```

When a document is rendered on [[GitHub]], is set to an empty value. These settings are used to ensure that the IDs generated by GitHub match the IDs generated by [[Asciidoctor]].

When remove it, could end up generating IDs that are invalid in DocBook output. Recommend either using a non-empty value or by [[Assign an explicit ID for a section]] for each section.
