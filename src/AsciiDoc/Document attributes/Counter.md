Is used to store and display ad-hoc sequences of numbers or Latin characters.

Is implemented as a specialized document attribute.

You declare and display a counter using an attribute reference, where the attribute name is prefixed with counter.

```
{counter:name}
```

Names follows the same as [[Rules for the user-defined attribute name]].

Is incremented and displayed every time the counter: attribute reference is resolved.

**increment** means to advance the attribute value to the next value in the sequence.
