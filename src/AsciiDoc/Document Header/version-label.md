You can set this attribute for change the `Version` word normally added at the start of the [[Revision Information]] by the [[AsciiDoc Processor]].
```
= The Intrepid Chronicles
Kismet Lee
v3: An icy winter incarnation
:version-label: Edition
```

You can unset this attribute for hide the `Version` word normally added at the start of the [[Revision Information]] by the [[AsciiDoc Processor]].
```
:version-label!:
```
