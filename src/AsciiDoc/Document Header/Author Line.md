Is directly after the [[Document Title]] line in the [[Document Header]].

Requires the following criteria to be correctly processed:
- The header must contain a [[Document Title]].
- The author information must be entered on the line directly beneath the document title.
- The author line must start with an [[author Attribute]] line.
- The content in the author line must be placed in a specific order and separated with the correct syntax.

[[middlename Attribute]] and [[email Attribute]] are optional.

```
= Document Title
firstname middlename lastname <email>
```

Accepts [[Multiple authors]].

```
= The Intrepid Chronicles
Kismet R. Lee <kismet@asciidoctor.org>
```
