Is assigned to three [[Built-in attribute]]s:

- [[revnumber Attribute]]
- [[revdate Attribute]]
- [[revremark Attribute]]

Can be assigned in a document header either by
- Using [[Revision Line]]
- Using [[Revision attribute entries]]

When default stylesheet applied, is displayed on the same line as the [[Author Information]] on rendering.

On rendering, is preceded with the word `Version`, added by the [[AsciiDoc Processor]]. Can be change or turned off with [[version-label Attribute]].
