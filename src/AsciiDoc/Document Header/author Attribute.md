represents the author’s full name.

It is composed by the following attributes:
- [[firstname Attribute]]
- [[middlename Attribute]]
- [[lastname Attribute]]
- [[authorinitials Attribute]]

When assigned via the [[Author Line]], includes all characters prior to the semicolon `;`, angle bracket (`<`) or the end of the line. Then, the Full Name can have a maximum of three space-separated names. If more, the full name is assigned to the [[firstname Attribute]].

You can adjoin names using an underscore `_` character.
