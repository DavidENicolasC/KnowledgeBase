The document’s revision number or version.

When assigned using the [[Revision Line]], must contain at least one number.

If it is not followed by a [[revdate Attribute]] or [[revremark Attribute]], it must begin with the letter `v`.
```
v7.0.6
```

Any characters preceding `v` are dropped on rendering.

If set with an [[Attribute Entry]], does not have to contain a number and entire value is displayed on rendering.
