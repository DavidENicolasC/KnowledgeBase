Consists of a name and a value separated by an = character
```
name=value
```

Rules for name are:
- Consists of a word character (letter or numeral)
- Is followed by any number of word or - characters

```
see-also
```

If value contains a space, comma, or quote character, it must be enclosed in double `"` or single `'` quotes.

If value contains quote character, it must be escaped by prefixing with a backslash
```
value="the song \"Dark Horse\""
```
