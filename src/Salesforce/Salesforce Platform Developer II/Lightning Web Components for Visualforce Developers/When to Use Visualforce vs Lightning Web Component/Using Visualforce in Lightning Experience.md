Visualforce pages are supported in many [[Low-code tools in Lightning Experience]].

Visualforce pages and Lightning Web Components can also coexist in the same page and interact.

To include a Lightning Web Component in a specific location on a Visualforce page, Use the Lightning Components for Visualforce Javascript Library. This works for both Aura components and Lightning web components.

To communicate among Lightning components and Visualforce pages in different hierarchies of the DOM, use Lightning Message Service.
