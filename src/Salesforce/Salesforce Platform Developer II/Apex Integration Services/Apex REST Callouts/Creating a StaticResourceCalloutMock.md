Here's the steps for create an Static Resource callout mock.
1. In the Developer Console, select **File** | **New** | **Static Resource**.  
2. For the name, enter the provided name.  
3. For the MIME type, select **text/plain**, even though we are using JSON.
4. Click **Submit**.
5. In the tab that opens for the resource, insert the content that the mock will return. This content is what the mock callout returns.
