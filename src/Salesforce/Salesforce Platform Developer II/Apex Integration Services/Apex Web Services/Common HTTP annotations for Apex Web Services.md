The following annotations corresponds to the [[Common HTTP Methods]]. You can use each annotation only once in an Apex Class.

| Annotation | Action | Details |
| @HttpGet | Read | Reads or retrieve records |
| @HttpPost | Create | Creates records |
| @HttpDelete | Delete | Deletes records |
| @HttpPut | Upsert | Typically update or creates records |
| @HttpPatch | Update | Typically update fields in records |
