1. You need to download and install [Android Studio from the Android Developers website](https://developer.android.com/studio)
2. Set the `JAVA_HOME` environment variable
3. Use the SDK Manager in Android Studio to install the following:
	- Android SDK Platform
	- Android SDK Platform-Tools
	- NDK (Side by side)
	- Android SDK Build-Tools
	- Android SDK Command-line Tools
4. Set `ANDROID_HOME` and `NDK_HOME` environment variables.
