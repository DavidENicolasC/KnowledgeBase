[[Framework]] for building tiny, fast binaries for all major desktop and mobile platforms.

 Is a polyglot and generic toolkit that is very composable and allows engineers to make a wide variety of applications. It is used for building applications for [[Desktop Computer]]s using a combination of [[Rust Tool]]s and [[HTML]] rendered in a [[Webview]].

Can integrate any frontend framework that compiles to HTML, [[JavaScript]], and [[CSS]] for building their user experience while leveraging languages such as [[Rust]], [[Swift]], and [[Kotlin]] for [[Backend]] logic when needed.

Uses web technologies that means that virtually any frontend framework is compatible.
